Create a user

To create a user, follow these steps:

  1. Go to the "Users" page.
  2. Click the "Create User" button.
  3. Fill out the user information form.
  4. Click the "Create User" button.

Delete a user

To delete a user, follow these steps:

  1. Go to the "Users" page.
  2. Select the user you want to delete.
  3. Click the "Delete User" button.
  4. Click the "Delete User" button again to confirm.

Update a user

To update a user, follow these steps:

  1. Go to the "Users" page.
  2. Select the user you want to update.
  3. Make the desired changes to the user information form.
  4. Click the "Update User" button.
